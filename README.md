# Обо мне

**Аналитик данных** с опытом работы с большими массивами информации и научной исследовательской деятельностью. Умею превращать сырые данные в бизнес‑инсайты: от выявления закономерностей до разработки прогнозных решений и визуализации результатов.

## Профессиональный путь

Начала карьеру в научной сфере, где развила ключевые аналитические навыки:
- работа с комплексными данными;
- выявление закономерностей;
- построение обоснованных выводов;
- эффективная визуализация результатов.

Постепенно расширила область применения этих навыков: перешла от поиска способов увеличить срок службы детали - к поиску способов увеличить LTV клиента. И там, и там цель одна - продлить "жизнь" системы, просто теперь система цифровая.
